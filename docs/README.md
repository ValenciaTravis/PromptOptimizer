# Docs Usage Guide

This directory stores reusable writing templates for the Prompt Optimizer development process.

## Files

- `dev-log-template.md`: Template for continuous development logs
- `acceptance-report-template.md`: Template for feature acceptance reports (F01-F04)

## How To Use

1. For daily implementation progress, copy `dev-log-template.md` structure into your working log file and append entries.
2. For each feature (F01/F02/F03/F04), create one report from `acceptance-report-template.md`.
3. Submit acceptance report first and wait for confirmation before moving to the next feature.
4. Keep all commands, test steps, and outputs reproducible.
